Light/Dark Theme Combos

Combo themes include two flavours — one light and one dark theme. They can be used on sites that support both, light and dark mode. By default the light theme will be displayed. Whenever an HTML element has a class that contains the word dark, all containing code blocks are rendered using the dark falvour.

__LIGHT_DARK__

Single Themes

Single themes just come in one single flavour. Unlike theme combos, a single theme is not affacted by classes of parent elements.
